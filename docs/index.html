How it works

A filter button lives directly in the playback bar.

Open the filter menu beside the volume control, select a look, and it applies immediately. Your selection is remembered in that browser. Filters run on the client GPU, preserving direct play and direct stream.

Why I made it

It started with Spider-Noir.

I wanted to watch Spider-Noir in black and white through my own Jellyfin setup instead of being limited to where an alternate presentation was available. That simple idea turned into a full collection of movie, black-and-white, color, comfort, and fun filters. It worked so well for me that I wanted to share it with every other Jellyfin user.

Installation

Install through Jellyfin's plugin catalog.

Playback Filters currently supports Jellyfin Server 10.11.x and requires the JavaScript Injector plugin.

  1. Install JavaScript Injector Add its 10.11 repository and install it first. https://raw.githubusercontent.com/n00bcodr/jellyfin-plugins/main/10.11/manifest.json
  2. Add the Playback Filters repository Open Dashboard → Plugins → Catalog → Settings and add: https://raw.githubusercontent.com/hdglifezach/Jellyfin-Playback-Filters/main/manifest.json
  3. Install and restart Install Playback Filters from the catalog, restart Jellyfin, and fully reopen Jellyfin Web.

Compatibility

Filters work in Jellyfin Web and web-based clients that load the server-provided interface. Native Roku, Kodi, and some television apps do not execute Jellyfin Web JavaScript and cannot display these filters.
