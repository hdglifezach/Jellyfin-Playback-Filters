# Playback Filters v1.1.0

## Changed

- Moved the filter button beside Jellyfin's standard volume control.
- Removed the placement dependency on the optional Jellysleep plugin.
- Added fallback placement for web clients with slightly different playback-bar markup.
- Added a public landing page with feature details, compatibility information, and installation steps.

## Existing Features

- More than 30 grouped real-time video filters
- Movie, Black & White, Color, Comfort, and Fun categories
- Animated Rainbow Shift
- Persistent per-browser filter selection
- Direct play and direct stream remain available

## Requirements

- Jellyfin Server 10.11.x
- JavaScript Injector
- Jellyfin Web or a web-based Jellyfin client
